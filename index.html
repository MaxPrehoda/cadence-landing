How It Works

01 / TRANSCRIPTION

Whisper.cpp Engine

OpenAI's Whisper running locally via C++ inference. Sub-200ms on Apple Silicon. Your voice never touches a server.

02 / INJECTION

System-Wide Control

CGEvent API for universal text insertion. Works in Slack, VSCode, terminal, anywhere you can type.

03 / POLISH

LLM Grammar Fix

Optional Qwen 2.5 1.5B (local) or Claude Haiku (cloud) for real-time grammar correction. Toggle on demand.

04 / COMMANDS

Custom Automation

Define voice shortcuts for apps, keyboard macros, or text snippets. "Open Figma" → launches instantly.

05 / ARCHITECTURE

Tauri + Rust + Svelte

Native performance, tiny binary (~8MB). Modern stack, easy to fork and extend. Zero Electron bloat.

06 / LATENCY

Instant Wake

CGEventTap hotkey monitoring. No polling, no lag. Press → record → transcribe → insert in under 300ms.
